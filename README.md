# Extract IB Data

## Project Goal
HTML-based tool for extracting and processing Interactive Brokers account data.

## Usage
Web application for extracting and analyzing IB data.

### Running the Application
Open the main HTML file in a web browser:
```bash
open index.html
```

### Features
- Interactive Brokers data extraction
- Data visualization
- Export functionality

### Requirements
- Modern web browser
- IB account data access
